グッドラムズ：サンコー下水部門年鑑

ERC721 | 200個限定、オーダーメイド、利回り付き、サンコー・チェーン上のDeFi融合PFP

グッドラムズ サンコー下水部門年鑑は、サンコー・チェーン上のNORMANCOMICSによる独占的なNFTコレクションで、200個のカスタムメイドでDeFiと統合されたPFP（プロフィール画像）が特徴です。各NFTは手描きで、利回りを提供し、所有者に独自のボーナスと特典を提供します。ミント後、コレクターは**@KEKBONDS**に連絡してPFPをカスタマイズできます。

🗂 プロジェクト概要

このNFTコレクションは、DeFi、ゲームFi、ソーシャルFiをマルチチェーンエコシステム内で融合させています。主な特徴は以下の通りです：

	•	限定ボーナス：HODLer向けの四半期配当とエアドロップ。
	•	年鑑所有者向けの特別特典：200個のNFTに限定され、それぞれがオーダーメイドのカスタマイズ付き。
	•	マルチチェーンでのミントと収益：DeFiおよびコレクタブル層全体での報酬をサポート。

💸 サンコー・チェーンでのマルチ通貨ミント

グッドラムズ サンコー下水部門年鑑コレクションは、独自のマルチ通貨ミントシステムを導入し、ユーザーがサンコー・チェーン上で主要通貨である**$DMT**とともに、さまざまなギャング関連通貨を使用して限定の利回り付きPFPをミントできるようにします。この柔軟性はアクセシビリティを高め、ユーザーが選択した通貨でコレクションに参加できるようにします。

	注： ギャング関連通貨の価格は市場の変動やギャング関連の税金により変更される場合があります。

	注： マルチ通貨ミントオプションは2025年第4四半期まで保留中です。

GOODLUMSはブロックチェーンが存在する前から始まり、元々は1990年代後半にNORMANCOMICS（@KEKBONDS）によって作成されました。彼らは紙のインクから始まり、建物、電車、橋、屋根のペイントへと進化し、最終的にはギャラリーショー、インディーズ🛹企業向けのスケートデッキ、（主に元ガールフレンドの）タトゥー、アンダーグラウンドヒップホップ、パンクアルバムカバーなどに至りました。

web9の発見により、プロジェクトはEthereumで開始され、Counterparty（$XCP）やBTCスタンプを介してBitcoinに拡大し、その後BaseやDegenに移行しました。そして最終的に、GOODLUMSは**サンコー・チェーン（$DMT）**に居場所を見つけ、真のデジェンコミュニティを引きつけました。さらに、隠されたスピンオフトークンやアセットが未公開のチェーン上でミントされ、熱心なユーザーによる発見を待っています。
